# 常见的角色(Role)示例

### (1)允许读取核心 API 组中 Pod 的资源：

```yaml
rules:
- apiGroups: [""]
  resources: ["pods"]
  verbs: ["get", "list", "watch"]
```

### (2)允许读写"extensions"和"apps"两个 API 组中的 deployment 资源

```yaml
rules:
- apiGroups: ["extensions", "apps"]
  resources: ["deployments"]
  verbs: ["get", "list", "watch", "create", "update", "patch", "delete"]
```
